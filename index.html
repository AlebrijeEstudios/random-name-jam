Random Name Jam

Making games starting with nothing but a random name!

What?

Recently, made some buzz on Twitter. So [@pstn_](https://twitter.com/pstn_), [@peterschueler](https://twitter.com/peterschueler) and [@lnwdr](https://twitter.com/lnwdr) came up with the idea of making actual games out of the random names from that site.

And of course, everone is welcome to join us!

Where?

Whereever you want. To keep organization simple, we won't meet in person.

When?

From 2015-03-23 00:00:00 to 2015-03-29 23:59:59.

The Rules

Anyone interested in participating should add their Github username to the list below via pull request.

Hit the EDIT button on [this page](https://github.com/lnwdr/random-name-jam/edit/gh-pages/index.html), copy the empty list entry and create a pull request.

When the jam starts, you will be given 3 random game names, from which you pick one and develop a game with that name.

You can pick any platform you like but of course it'd be nice, if as many people as possible can actually play sour game.

When the deadline comes, there will be a new list here for the finished games. Again, add yours via pull request, just as before.

This is not a contest. There will be no winner, no prize, just games.
